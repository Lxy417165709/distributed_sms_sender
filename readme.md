## 简介
这是一个分布式的短信发送器(短信发送没实现)。实现了一段时间内，同一用户最多发送 `max` 次短信的限制。

## 技术栈
`Go` + `Redis` + `Docker`

## 特点
1. 同一用户可位于不同机器。
2. 后端以 `Go` 集群形态为用户服务。
3. 使用 `Redis` 实现分布式锁。
4. 使用 `Redis`集群  + 一致性哈希算法 实现分布式缓存。
5. 使用 `Nginx` 实现负载均衡。
5. 代码优美。  `:)`
